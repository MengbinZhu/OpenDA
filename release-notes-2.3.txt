OpenDA 2.3 Release Notes.

June 2016

This is a release containing bug fixes, enhancements and new functionality. For an overview of the changes in version 2.3, see Change-log-2.3.txt.

For installation instructions and the user manual, visit www.openda.org or view the 'doc' folder.

New Features:

 - Distributed Evaluation of Local Sensitivity Analysis (DELSA) algorithm
 
 - Delft3D Black Box wrapper
 
 - Sobek3 Black Box support

 - strict boundaries for the sparseDud algorithm

Known issues:
 
 - BMI models will not function properly with this release
